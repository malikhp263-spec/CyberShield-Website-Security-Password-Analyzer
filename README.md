Project Description

CyberShield is a web-based cybersecurity application designed to help users evaluate website security and analyze password strength. The system provides a simple and user-friendly interface for checking whether a website is secure and determining the strength of passwords based on various security factors.

---

Objectives

- To promote cybersecurity awareness.
- To identify secure and insecure websites.
- To evaluate password strength.
- To encourage the use of strong passwords.
- To provide a simple security assessment tool for users.

---

Features

Website Security Checker

- URL Validation
- HTTPS Detection
- Website Security Status
- Security Score Evaluation
- User-Friendly Interface

Password Strength Analyzer

- Password Length Analysis
- Uppercase Character Detection
- Lowercase Character Detection
- Number Detection
- Special Character Detection
- Weak, Medium, and Strong Classification

---

System Workflow

1. User enters a website URL.
2. The application checks whether the URL is valid.
3. HTTPS security is verified.
4. Security status is displayed.

For Password Analysis:

1. User enters a password.
2. The system checks length, uppercase, lowercase, numbers, and symbols.
3. A security score is generated.
4. Password strength is displayed.
